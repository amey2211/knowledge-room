# Knowledge Room: Study Partner and Discussion Web-app
Knowledge Room is a modern, responsive web application designed to help learners find study partners, join topic-based rooms, and collaborate through discussions. The platform provides a clean and intuitive interface where users can browse topics, explore active rooms, and participate in conversations in real time. 
## Key Features
### 1. Topic-based Browsing
Users can explore rooms categorized under Python, Backend Development, Frontend Development, and more. Each topic displays the number of active rooms and participants.

### 2. Room Creation & Participation
Registered users can create new study rooms, join existing ones, and engage with others through posts and replies.

### 3. Recent Activities Feed
A real-time sidebar shows the latest interactions, who replied, what they said, and when to keep users updated on ongoing discussions.

### 4. User Authentication
Includes a login system with email and password support, ensuring secure access to personalized features.

### 5. Clean & Modern UI
Built with a dark-themed, elegant design featuring a smooth layout, iconography, and well-structured navigation for a better user experience.

## Purpose
The project aims to create a collaborative learning environment where people can connect, share knowledge, and enhance their skills together, especially helpful for students, self-learners, and online study groups.
